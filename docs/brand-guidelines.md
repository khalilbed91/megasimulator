**MegaSimulator — Brand Guidelines (brief)**

Purpose: establish a clear identity for the app (colors, typography, logo usage).

Palette
- Primary (Teal): #0EA5A4 — main accent, CTA, highlights
- Dark neutral: #071022 — dark background / surfaces
- Light neutral: #F4F6F8 — page background (light theme)
- Success green (inactive tab): #2E7D32
- Danger red (active tab): #C62828

Typography
- UI font: Inter (system fallback: system-ui, Arial)
- Headings: 600 weight; Body: 400
- Scale: H1 28px, H2 20px, Body 14-16px

Logo
- Use the square mark (logo.svg) on app chrome and favicons.
- For dark backgrounds prefer white-on-accent; for light backgrounds prefer accent mark on white.

Do's
- Keep generous spacing around the logo (at least 16px)
- Use primary for CTAs, red only for active tab state

Don'ts
- Don't stretch or recolor the logo arbitrarily
- Don't mix multiple accent colors in a single component

Accessibility
- Ensure contrast of text vs background meets WCAG AA (use dark text on light surfaces or light text on dark surfaces).
